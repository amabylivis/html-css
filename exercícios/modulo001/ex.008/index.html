Principais formatações

Negrito ou destaque

Nessa frase temos um termo em negrito usando a tag < b > < /b > (não semântica)

Nesta frase temos um termo em destaque usando a tag < STRONG > (em semântica)

Itálico ou ênfase

Nesta frase temos um termo em itálico usando a tag < i > < /i > (não semãntica)

Nesta frase temos um termo em ênfase usando a tag < EM > (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> (semântica)

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno aqui

Usando a tag < small > e < big > (obsoleto)

Texto deletado

Podemos marcar um texto excluído para indicar que ele deve ser lido mas não considerado, usando a tag < dell >

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois, utilizando a tag < ins >

Existe também o sublinhado com a tag < U > (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, utilizando a tag < sup >

Texto subescrito

Para inserir coisas do tipo H2O, utilizando a tag < sub >
